# 創発ルール一覧

箱庭に配置されたオブジェクトの隣接パターンによって自動的に発生する視覚エフェクト。  
描画時にグリッドをスキャンして判定する（DBへの書き込みはなし、純粋にレンダリング側の演出）。

実装: `src/components/GardenEmergence.ts`

---

## クラスタ系（同種オブジェクトの連結判定）

### 1. 町 (Town)
- **条件**: house が3つ以上隣接（4方向で連結）
- **効果**: 
  - 家の間の地面が舗装される（明るい石畳タイル）
  - 密集地点（隣接 house が2つ以上）にランタンポストが出現
- **意図**: 家が並ぶと「ただの建物」から「町」に見え始める

### 2. 夜市 (Night Market)
- **条件**: house が4つ以上隣接
- **効果**: 密集した家にカラフルな屋台風の庇（awning）が出現
- **意図**: 町がさらに栄えると夜市感が出る。赤・黄・緑・青の庇が祭り感を演出

### 3. 森 (Forest)
- **条件**: tree が3つ以上隣接
- **効果**:
  - 地面に苔テクスチャ（半透明の暗緑プレーン）
  - 下草のブッシュメッシュ
  - 偶数座標にキノコ（赤い傘＋白い軸）
- **意図**: 木が集まると「森」になる。足元の密度で森感を出す

### 4. 花畑 (Flower Field)
- **条件**: flower が3つ以上隣接
- **効果**: 地面に小さな花のドット（6個/セル）が円形に散布
- **意図**: 花が集まるとカーペット状に広がる。色は元の花カラーパレット準拠

## 隣接ペア系（異種オブジェクト間の反応）

### 5. 庭園邸宅 (Garden Estate)
- **条件**: house の隣（4方向）に flower がある
- **効果**: house の周囲4辺に生垣（低い緑のボックス）が出現
- **意図**: 家の横に花 → 庭付きの邸宅になる

### 6. 街路樹 (Street Tree)
- **条件**: tree の隣（4方向）に house がある
- **効果**: tree の斜め方向にランプポスト（暖色白の街灯）が出現
- **意図**: 住宅街の街路樹 → 街灯が灯る

### 7. 塔の森 (Tower Grove)
- **条件**: tower の隣（8方向）に tree が2つ以上ある
- **効果**: tower の壁面にツタ（緑のボックス）が這い上がる
- **意図**: 自然に飲まれた塔、廃墟ファンタジー感

### 8. 花時計 (Flower Clock)
- **条件**: flower の隣（4方向）に tower がある
- **効果**: flower の足元に光るトーラスリング（acid lime）
- **意図**: タワーの足元を花が飾るモニュメント感

### 9. スカイブリッジ (Sky Bridge)
- **条件**: tower 同士が隣接（4方向）
- **効果**: 2つのタワー間に渡り廊下メッシュが架かる（高さ 1.1）
- **意図**: 高層ビル群 → 未来都市のスカイブリッジ

## 特殊条件系

### 10. 神木 (Sacred Tree)
- **条件**: tree が孤立（8方向全てに何もない）かつ growth_stage > 1.5
- **効果**:
  - 足元に鳥居（赤い柱2本 + 横木）
  - 周囲に金色の光粒リング（12個のポイント）
- **意図**: 孤高で大きく育った木 → 御神木。成長しないと発動しない特別感

### 11. 並木道 (Avenue)
- **条件**: tree が同じ行 or 列に3つ以上連続
- **効果**: 隣接する tree 間の中間地点に提灯（光るスフィア）が浮かぶ
- **意図**: 整列した木 → 参道・並木道の提灯

### 12. 広場 (Plaza)
- **条件**: 空きセルの4方向に3つ以上のオブジェクトがあり、かつ2種類以上
- **効果**: 空きセルに噴水が出現（台座 + 柱 + 水面）
- **意図**: 賑やかなエリアの中心にある広場・噴水

## 新オブジェクト連動系

### 13. 麦畑 (Wheat Field)
- **条件**: windmill の隣（4方向）に flower がある
- **効果**: 風車の足元に金色の麦畑テクスチャ + 麦の茎
- **意図**: 風車 + 花 → 田園風景の完成

### 14. 精霊火 (Spirit Orbs)
- **条件**: shrine の隣（8方向）に tree がある
- **効果**: 神社の周囲に翡翠色の光球が浮遊
- **意図**: 森の中の神社 → 霊的な雰囲気

### 15. 灯りの小路 (Lit Walkway)
- **条件**: lamp 同士が隣接（4方向）
- **効果**: ランプ間の地面に暖色の光の帯
- **意図**: 街灯が並ぶと散歩道になる

### 16. 映り込み (Reflection)
- **条件**: pond の隣（4方向）に statue がある
- **効果**: 池の水面に金色のきらめきドット
- **意図**: 像が水面に映り込む幻想的な景観

### 17. 記念碑の光柱 (Monument Aura)
- **条件**: statue が孤立（8方向に同種なし）かつ growth > 1.0
- **効果**: 像の周囲に半透明の光柱リング（6本）
- **意図**: 長く育った孤高の像 → 神々しいモニュメント

## 設計メモ

- 判定は全てフロントエンドで描画時に行う。DB や Rust 側には影響しない
- 凍結ギャラリーでも同じルールが適用される（スナップショット時の配置で再判定）
- 成長によるスケーリングは創発エフェクトには影響しない（ベースオブジェクトのみ拡大）
- ルールの実行順序: 地面エフェクト → デコレーション → 特殊条件
- 複数ルールが同一セルに適用されることがある（町 + 夜市、街路樹 + 庭園邸宅 など）
